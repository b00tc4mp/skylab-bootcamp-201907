Esto es el juego llamado Pasapalabra.

El juego consiste en responder a cada enunciado con la palabra que crees que mejor encaja en la definición presentada.

El sistema de puntuación es el siguiente:

  • Cada letra acertada es un punto.
  • Si fallas una palabra no se vuelve a repetir y cuenta como fallo
  • Si pones pasapalabra o la letra p, no cuenta como fallo y se queda pendiente para el siguiente turno.
  • Gana el que más palabras haya acertado
